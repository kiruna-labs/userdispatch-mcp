# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in UserDispatch, please report it responsibly.

**Email:** [security@kiruna.ai](mailto:security@kiruna.ai)

Please include:

- A description of the vulnerability
- Steps to reproduce
- Potential impact
- Suggested fix (if any)

We will acknowledge your report within 48 hours and aim to release a fix within 7 days for critical issues.

## Do Not

- Open public GitHub issues for security vulnerabilities
- Exploit vulnerabilities against production systems
- Access or modify other users' data

## Security Documentation

For a full overview of UserDispatch's security architecture — including authentication, rate limiting, input validation, and data privacy — see our [Security Docs](https://userdispatch.com/docs/security).
